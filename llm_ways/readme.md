### API GigaChat
Для работы с проектом API GigaChat вам понадобятся файл gigachat_api.py config.py и как пример результата report_20260123_1757.txt. <br>
Чтобы запустить GigaChat нужно <br>
1) получить ключ authorization_key с официального сайта API GigaChat;<br>
2) вставить ключ в файл **config.py**; <br>
3) подготовить текстовые данные для обработки (могу поделиться); <br>
4) отредактировать путь до текстового файла и стобца обработки файла, если это excel-таблица; <br>
5) запустить скрипт в файле **gigachat_api.py** <br>
6) Файл **report_20260123_1757.txt** - пример результата
