# Поиск топ тем <br>
Нужно найти n-ое количество топ тем разными способами. <br>
1) В папке ml_algos находятся пайтон файлы с алгоритмами машинного обучения <br>
   а) kmeans,<br>
   б) dbscan. <br>
Это классические алгоритм для решения задачи кластеризации. <br>

2) В папке llm_ways представлены <br>
   а) скрипты LLM на локальном сервере Ollama, <br>
   б) HTTP API GigaChat'а. <br>
LLM качественней справляется с сзадачей поиска тем из неструктурированного массива текстовых данных. <br>

Дисклеймер <br>
все скрипты написаны с применением ИИ! <br>

Текстовых данных над которыми прозводятся операции могу предоставить по запросу. <br>
