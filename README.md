# top_topics
Нужно найти n-ое количество топ тем разными способами. 
1) В папке ml_algos находятся пайтон файлы с алгоритмами машинного обучения
   а) kmeans,
   б) dbscan.
Это классические алгоритм для решения задачи кластеризации.

2) В папке llm_ways представлены /n
   а) скрипты LLM на локальном сервере Ollama,
   б) HTTP API GigaChat'а. 
LLM качественней справляется с сзадачей поиска тем из неструктурированного массива текстовых данных.

Дисклеймер
все скрипты написаны с применением ИИ! 

Текстовых данных над которыми прозводятся операции могу предоставить по запросу.
